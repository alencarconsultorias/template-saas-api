# template-saas-api

## ⚙️ Configuração do Ambiente 
1. Identificar o arquivo `docker-compose.yml` no projeto. 
2. Instalar o docker e subir o compose: 
```bash
docker-compose up -d 
```
3. Apos instalacao, identificar o container criado: 

```bash
docker ps 
```
